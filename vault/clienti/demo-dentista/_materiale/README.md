# _materiale — Studio Dentistico Sorriso

Il materiale grezzo da cui è stato ricavato il vault: export delle vecchie chat WhatsApp,
appunti della call, recensioni, listino, screenshot.

Non lo legge nessun sistema. Serve a noi per rifare o correggere la curatela senza ripartire
da zero, e per dimostrare al cliente da dove viene ogni risposta.

**Tenant di test: qui non c'è materiale reale.**
